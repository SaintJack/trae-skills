# User Rule Usage Example

下面这段内容适合粘贴到：

- `Trae > Settings > Rules > User Rules`

来源文件：

- `rules/user/engineering-gate-global.md`

推荐场景：

- 你希望所有项目默认都先做需求对齐
- 你希望 AI 在技术方案阶段必须向你申请批准
- 你不希望 AI 在需求未清时抢跑执行

推荐配合方式：

1. 将全局用户规则粘贴到 `User Rules`
2. 在具体项目中安装 `Skill + Project Rule`
3. 如有需要，再手动创建 `Engineering Gate Agent`

这样可以形成：

- 全局行为约束
- 项目内规则增强
- Agent 级工作流模板
